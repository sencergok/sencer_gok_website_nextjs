# Sencer Gök - Kişisel Portfolyo Sitesi

Bu proje, mobil uygulama geliştiricisi Sencer Gök'ün kişisel portfolyo sitesidir. Next.js, Tailwind CSS ve Shadcn UI kullanılarak geliştirilmiştir.

## Özellikler

- Modern ve duyarlı tasarım
- App Store'da yer alan projelerin sergilenmesi
- İletişim formu
- Sosyal medya bağlantıları

## Kullanılan Teknolojiler

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn UI
- React Hook Form

## Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/yourusername/sencer_gok_website_nextjs.git
```

2. Bağımlılıkları yükleyin:
```bash
cd sencer_gok_website_nextjs
npm install
```

3. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

4. Tarayıcınızda http://localhost:3000 adresini açın.
